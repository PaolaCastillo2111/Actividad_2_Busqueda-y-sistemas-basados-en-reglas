# S.I de Rutas TransMilenio

Proyecto de Inteligencia Artificial - Búsqueda y sistemas basados en reglas - grafos y búsqueda heurística mediante el algoritmo A*.

## Descripción

El sistema encuentra la mejor ruta entre portales de TransMilenio utilizando técnicas de Inteligencia Artificial.

## Tecnologías utilizadas

- Python
- Grafos
- Algoritmo A*
- Búsqueda heurística
